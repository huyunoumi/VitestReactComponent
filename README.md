# VitestReactComponent
Practice Vitest for React's component with [サバイバルTypeScript](https://typescriptbook.jp/tutorials/vitest-component-test).

# Initialization
1. install dependencies
```shell
npm install \
  react \
  react-dom \
  @types/react \
  @types/react-dom
```
2. install debug dependencies
```shell
npm install -D \
  typescript \
  vitest \
  vitest-browser-react \
  @vitejs/plugin-react \
  @vitest/browser-playwright
```
